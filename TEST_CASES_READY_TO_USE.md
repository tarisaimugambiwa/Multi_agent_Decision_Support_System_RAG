## üéØ How to Use These Test Cases

### Step 1: Create or Select Patient
- Go to Patients section
- Create new patient or select existing test patient
- Enter demographic information as provided

### Step 2: Create New Case
- Click "New Diagnosis Case"
- Select the patient
- Copy-paste symptoms exactly as written above
- Enter vital signs section by section
- Copy-paste medical history

### Step 3: Submit for Diagnosis
- Click "Generate AI Diagnosis"
- Wait for multi-agent system to process (5-10 seconds)

### Step 4: Review Results
- Check confidence score matches expected range
- Review primary diagnosis
- Check if treatment recommendations are appropriate
- Verify urgency level classification
- Review knowledge base sources cited

### Step 5: Test Regenerate Feature (Optional)
- Click "Regenerate Diagnosis" button
- Compare results with original diagnosis
- Confidence should remain similar
- Treatment plan should be consistent

## üîß Troubleshooting

### Low Confidence (<70%)?
- Add more specific symptom details
- Include more vital signs
- Add temporal information (duration, progression)
- Include relevant medical history
- Click "Regenerate Diagnosis"

### Wrong Diagnosis?
- Check if symptoms match knowledge base categories
- Ensure symptoms are medically accurate
- Add differential symptoms to clarify
- Review spelling and medical terminology

### Treatment Plan Too Generic?
- This is expected for conditions not well-covered in knowledge base
- System will provide standard clinical care protocols
- Review diagnosis explanation for AI reasoning
